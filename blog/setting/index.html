Ubuntu 한글 설정

By Jerry Kim 20 August 2016

우분투를 설치를 했으나!!!! 한글 키보드를 설정을 해야합니다. 그럼 시작해볼까요. (이번에도 사진 촤라라락..)

1. 시스템 한글 설정

부팅을 했습니다. 크으… 깔끔깔끔…

오른쪽 위에 전원 버튼 누르고
System Settings로 들어가서 Language Support를 눌러줍니다.

그럼 설치를 해야한다고 하네요. 묻지도 따지지도 말고 설치합시다. (뒤에 사진찍은건 애교로 넘어갑시다…)

로그인 비밀번호 입력하구요. (여기도 있네…)

그럼 설치가 시작됩니다.

설치가 끝나면 한국어라고 똿! 나옵니다.

아래에 있는 한국어를 드래그해서 맨 위로 올려줍니다.

그리고 ‘Apply System-wide’를 클릭!

비밀번호 입력하시구요.

Regional Formats는 설치할때 이미 Seoul로 해서 상관은 없지만 그냥 들어가서 ‘Apply system-wide’눌러줍니다.

로그아웃을 하고 다시 로그인 해주세요!! 그럼 다음과 같이 창이 뜨는데 저는 영문이 편해서 그냥 이름을 유지하기로 했습니다.

2. 한글 키보드 설정하기

키보드를 설정해보죠.
시스템 설정으로 들어갑니다.
그리고 텍스트 입력창을 눌러줍니다.

다음과 같이 뜨는데요. 저기에 +를 눌러줍니다. (정말 더럽게 못그리네..)

쭉쭉쭈욱~ 내려서 ‘한국어 (Hangul)(IBus)‘를 선택하고 ‘추가’버튼을 누릅니다.

지금은 키 변환이 Super+스페이스로 되어있네요. 이대로 쓰실분들은 쓰셔도 됩니다. 하지만!! 저는 한/영 키를 사용하고 싶기 때문에… 저 키를 바꿔보도록 하죠. ‘키보드 설정…‘을 누릅니다.

자판입력 탭에서 구성키를 ‘오른쪽 Alt’로 바꾼 후에 ‘다음 입력 소스로 전환’ 키는 한/영으로 바꿉니다.

다시 텍스트 입력으로 돌아와서 한국어를 선택한 후 사진에 표시된 도구 버튼을 누릅니다.

다음과 같이 나오는데요.
‘한글 모드로 시작’을 체크한 후 확인을 눌러줍니다.

이제 한글에 대한 설정이 끝났습니다!
다시 로그아웃을 한 후 로그인을 하시면 한글을 쓰실 수 있습니다!!

오늘은 포스팅 Day인듯합니다. 왜냐하면 오늘 우분투를 재설치 했기 때문이죠… 으하하하하하하하핳…… 우분투 다시 셋팅하면서 포스팅중…. 다음은 소프트웨어 관련 세팅 포스팅하겠습니다.
